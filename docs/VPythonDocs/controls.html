Widgets: buttons, sliders, menus

In classic VPython, the example shown above, ButtonsSlidersMenus-VPython, would use wxPython to display buttons and sliders. In the browser environment, this has to be done in a different way. Run the program and inspect the program code to see how to create buttons, sliders, and drop-down menus to control the execution of a 3D animation.

You'll see dollar signs ("$") all through the example. It is a standard abbreviation for jQuery, a widely used library for making connections between a JavaScript program and the HTML of a web page. You may wish to study a jQuery tutorial. GlowScript imports the jQuery and jQuery-ui libraries.

Full understanding of the jQuery mechanisms requires a good knowledge of HTML coding as well as knowledge of jQuery itself. However, the examples shown in the program above should provide you with enough information to be able to use controls in your own GlowScript programs.

Important: Do not use character strings to specify a function as an argument in a jquery statement, as the character string is evaluated in a global context outside the GlowScript context, and your function will not be found. For example, say something like $("<button/>").click(handle_click) rather than something like $("<button onclick='handle_click'></button")

It is common practice in JavaScript programs and jQuery statements to use what are called "anonymous" functions, and the RapydScript-to-JavaScript compiler extends Python to permit the use of such functions. Consider the following code, which binds the function "process_click" to a click event, so that when you click, the function is executed:

def process_click():
    print('You clicked.')

scene.bind('click', process_click)

Now consider the equivalent code in which the function is "anonymous":

scene.bind('click', def():
    print('You clicked.')
)

As usual, the body of the function must be indented.
